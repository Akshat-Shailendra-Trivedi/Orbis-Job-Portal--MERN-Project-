# Orbis â€“ Job Portal Web Application (MERN Stack)

Orbis is a full-stack Job Portal web application designed to simplify job searching and recruiting.  
The platform allows users to create accounts, apply for jobs, manage profiles, and track applications, while recruiters can post jobs and manage applicants.  
Built using the **MERN Stack**, Orbis focuses on real-world functionality, clean UI, and secure, scalable backend architecture.

### **For Recruiters**
- Create job postings  
- Edit and delete job listings  
- View applicant details

### **General Features**
- Secure authentication (JWT + bcrypt)  
- Fully responsive UI using React + Tailwind  
- RESTful API architecture  
- MongoDB database with schema validation  
- Clean folder structure and reusable components
